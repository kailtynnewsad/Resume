Color

Contrast

Contrast is a very important part of color selection in website design. Contrast can help reduce eyestrain and focus user eyesight on certain elements on the web page. The most important contrast is text color to background color. You want the user to be able to read the text clearly without the need to strain their version to read the text. The best practice is to light color for the background and dark color for the text. Contrast can also help draw the user to a certain part of the page. For example, having the main contents of the page in a lighter color draws the focus to that part of the page.

Contrast is also important when it comes to images on the page. You want the images on the web page to stand out and not fade into the background. The color of the background should not affect the user’s ability to see the images on the page. However, you also want to make sure the text is not overpowered by the images. You need to make sure the user can read the page clearly. Also, you need to make sure the user’s does not strain their eyes while looking at a website.

Color Psychology

Colors can have an effect on the user’s emotions. The color psychology rules can help you focus on what message and feelings to want to convey to the user. For example, red is bold color that has strong emotion to it and could make some feel a sense of urgency. Blue is associated with sense of security and trust. Which if you have noticed companies like to use the color blue often because it evokes those types of emotions. However, some colors should be considered carefully when choosing them.

Each color of as a different meaning or emotion. Red has a strong emotion and a sense of urgency. While orange means cheerful but sometimes means caution. Yellow is attention grabbing and means optimism. Green represents growth, nature, health, serenity, and wealth. Blue has an emotion of calmness and trust. Turquoise represents healing. The color purple represents royal and wealth. Brown represents nature and the Earth. The color black represents sophisticated and sleek. Gray represents security, reliability, and intelligence. The color white represents neutral.

Vibrancy

The vibrancy of color can dictate different emotions. Darker colors can relax and allow the user’s mind to focus on other things. Brighter colors can make the user feel energetic. Which can help when the website is advertising a certain product. Sites that have gossip or tabloid news often use bright colors like bright reds. Sites with important information or news use darker because those sites enquire more focus. Also, paler colors will be used for important information.

There are many different types of websites out for creating certain types of colors or testing what colors go together. These websites can help make color selection extremely easy and offer the whole color wheel at your disposal. They also can help test the vibrance of a certain shade of color. Some that are highly recommended are Kuler and Color Scheme Designer 3 (Paletton).
